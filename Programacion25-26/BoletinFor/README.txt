Boletín de prácticas: Iterador for (soluciones básicas)

01_contar.py                      -> Imprime del 1 a n (validando n > 0)
02_sumar_1_a_5.py                 -> Suma 1..5
03_sumar_1_a_n.py                 -> Suma 1..n (si n <= 0, finaliza)
04_pares_2_a_20.py                -> Pares 2..20
05_serie_0_a_20_pasos_de_2.py     -> Serie 0..20 de 2 en 2
06_contar_atras_10_a_1.py         -> Cuenta atrás 10..1
07_tabla_del_3.py                 -> Tabla del 3 (1..10)
08_tabla_de_un_numero.py          -> Tabla del número leído (1..10)
09_cuadrados_1_a_10.py            -> Cuadrados 1..10
10_multiplos_de_5_5_a_50.py       -> Múltiplos de 5 (5..50)

Todos los programas usan 'for' de forma sencilla y código muy básico.
